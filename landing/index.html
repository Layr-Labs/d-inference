The Thesis

AI inference shouldn't require trust

Cloud inference asks you to trust a corporation with your prompts, your data, your IP. We're building the alternative.

01

End-to-end encrypted

Your prompts are encrypted with the provider's X25519 public key before they leave your machine. The coordinator routes but never reads. NaCl box construction.

02

Hardware-attested

Every node proves identity through Apple Secure Enclave. P-256 keys in tamper-resistant silicon. Apple Enterprise Attestation CA chains verified by the coordinator.

03

Hardened runtime

Anti-debug (PT_DENY_ATTACH), binary self-hash, SIP enforced, Hypervisor.framework memory isolation. The Mac owner cannot access inference memory.

04

Decentralized by design

No cloud vendor lock-in. No GPU rental queues. A global mesh of Apple Silicon machines scored by decode speed, trust level, and hardware health.

Architecture

How it works

Three steps. No GPU rentals, no cloud accounts, no data retention.

01

Send a request

Use the OpenAI-compatible API. Your prompt is encrypted with the provider's public key before it leaves your machine. The coordinator never sees plaintext.

02

Matched to a verified node

The coordinator scores providers by decode speed, trust level, thermal state, and memory pressure — then routes to the optimal Apple Silicon machine.

03

Get your response

Inference runs in a hardened, isolated process with anti-tamper protections. Results stream back encrypted. The node operator cannot see your prompt or output.

"The question isn't whether AI inference will decentralize — it's whether the decentralized network can be trusted. That's the hard problem. That's what Lattice solves."
For Developers

One line to switch

Drop-in replacement for the OpenAI API. Change your base URL.

python 
from openai import OpenAI

client = OpenAI(
    base_url="https://inference.eigencloud.xyz/v1",
    api_key="your-api-key"
)

response = client.chat.completions.create(
    model="qwen3-30b-a3b",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")
