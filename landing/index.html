How it works

Three steps. No GPU rentals, no cloud lock-in.

Send a request

Use the OpenAI-compatible API. Your prompt is encrypted with the provider's public key before it leaves your machine. The coordinator never sees plaintext.

Matched to a verified Mac

The coordinator scores providers by decode speed, trust level, hardware health, and thermal state -- then routes to the best available Apple Silicon machine.

Get your response

Inference runs in a hardened, isolated process with anti-tamper protections. Results stream back encrypted. The Mac owner cannot see your prompt or output.

Verify, don't trust

Every layer gives you proof, not promises.

Hardware Verified

Every Mac proves it's genuine via Apple Secure Enclave. P-256 keys generated in tamper-resistant silicon.

End-to-End Encrypted

Prompts encrypted with provider's X25519 key using NaCl box. The coordinator routes but never reads.

Apple-Signed Attestation

Device certs from Apple's Enterprise Attestation CA. MDM SecurityInfo cross-check validates hardware identity.

Hardened Process

Anti-debug, binary self-hash, SIP enforced. No memory access to prompts or outputs -- even by the Mac owner.

For developers

Drop-in replacement for the OpenAI API. Change one line.

python 
from openai import OpenAI

client = OpenAI(
    base_url="https://inference-test.openinnovation.dev/v1",
    api_key="your-api-key"
)

response = client.chat.completions.create(
    model="qwen3-30b-a3b",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")
